# Product Context

## Why Nura Exists

Nura addresses a critical but under-recognized problem in software engineering: **psychological barriers in technical onboarding** disguised as documentation issues. The real challenge isn't missing documentation - it's the **fear-based culture** that prevents developers from asking questions.

### Root Cause Analysis

**Industry Problem**: Tech meritocracy creates "impostor syndrome amplification"
- Developers avoid questions to prevent appearing incompetent
- "Failure is not allowed" culture in competitive tech environment
- Social pressure to appear competent = learning paralysis

**Business Impact**: $150K+ per developer/year in sub-optimal productivity during 2+ week ramp-up periods
