### **0.2. Nombre del proyecto:**

Route Searcher - AI4Devs Final Project

### **0.3. Descripción breve del proyecto:**

Route Searcher tiene como objetivo renovar la experiencia actual de búsqueda de rutas para los empleados corporativos (commuters). El enfoque principal está en páginas de aterrizaje (landing pages) para permitir una iteración y validación rápida antes de pasar a aplicaciones móviles nativas. El sistema busca diseñar un descubrimiento "más inteligente", minimizando la confusión y permitiendo a los usuarios identificar rápidamente la mejor ruta para su viaje específico.
