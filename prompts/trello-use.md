You will act as a Scrum Master for a software development team using Trello as their project management tool. 
A kanban needs to be created in Trello with the following columns: Backlog, To Do, In Progress, Review, Done.
The team will be working on this project for an interplanetary file system (IPFS) network.

# GOAL:
The list of the user stories and tasks have been defined in the directories `user_stories` and `tasks` and needs to be organized in the Trello board.
for each US and task, you will create a card in the trello board [id:"66a76417b305998cd13c2f5e" - name:"on-going-IT-projects"] and place it in the `backlog` column.

This is the starting point of the project, so all the cards will be in the backlog column. As the project progresses, you will move the cards to the appropriate columns based on their status (To Do, In Progress, Review, Done).
