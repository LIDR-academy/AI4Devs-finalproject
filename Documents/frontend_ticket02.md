FRONTEND — Ticket 2

Título: Catálogo (Listado + Filtros + Paginación vía HTMX)

Descripción:
Construir la página de catálogo con búsqueda, filtros (jugadores obligatorio; tiempo/edad/categoría opcionales), paginación optimizada para móvil y actualizaciones parciales con HTMX.

Criterios de Aceptación:

Barra de búsqueda (enter/botón) y limpieza rápida.

Filtros con estado visible y accesibles (teclado/lectores).

Paginación clara (prev/sig, página actual).

“Sin resultados” con sugerencia de limpiar filtros.

HTMX: swaps parciales para filtros y paginación (sin recargar toda la página).

Performance UX: listado inicial < 3 s; interacciones HTMX < 2 s.

Diseño respeta Figma (grid/cards, espaciados, tipografía).

Prioridad: Alta
